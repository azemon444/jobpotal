# Job Portal Website

A modern, full-featured web application built with Laravel for connecting employers and job seekers. The platform supports job posting, application management, user profiles, and admin moderation, providing a seamless recruitment experience for all roles.

## Features

- User registration and authentication (Admin, Employer, Job Seeker)
- Employers can create company profiles, post jobs, and manage applications
- Job seekers can create/edit profiles, search jobs, and apply online
- Admin dashboard for user management, job post approval, and analytics
- Real-time notifications and alerts
- Role-based dashboards and permissions
- Advanced job search and filtering
- Responsive design for desktop and mobile
- Secure password management and profile editing

## Technologies Used

- Laravel (PHP framework)
- Blade templating engine
- MySQL database
- Bootstrap & custom CSS
- JavaScript & jQuery
- Spatie Permission package (role management)
- SweetAlert (user notifications)
- PHPUnit (testing)

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/azemon444/jobpotal.git
   cd jobpotal
   ```
2. **Install dependencies:**
   ```bash
   composer install
   npm install && npm run dev
   ```
3. **Copy and configure environment:**
   ```bash
   cp .env.example .env
   # Edit .env with your database and mail settings
   ```
4. **Generate application key:**
   ```bash
   php artisan key:generate
   ```
5. **Run migrations and seeders:**
   ```bash
   php artisan migrate --seed
   ```
6. **Start the development server:**
   ```bash
   php artisan serve
   ```

## Usage

- Register as a job seeker or employer and complete your profile.
- Employers can post jobs and manage applications from their dashboard.
- Job seekers can search for jobs, apply, and track their application status.
- Admins can approve/reject job posts, manage users, and view analytics.
